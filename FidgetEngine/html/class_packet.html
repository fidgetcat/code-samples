Detailed Description

Wrapper for sfml Packet class that provides primitive Packet generation and manipulation.

StuffMe methods place data of the appropriate type into the packet. They can be placed into outside variables using pointers together with PopMe methods. Variables must be popped out in the same order they were stuffed in.
