Detailed Description

Container/linked list used to manage Objects.

Mainly used by CObjects. Manages nodes that store pointer information.

ObjLsNode * ObjCollection::Add ( Object *  input,
Transform  inTrans 
)

This list adds nodes to the start of the list, not to the end, with the oldest nodes at the end of the list. The pointer returned is used by the Object associated with this node.

Parameters
input- The object associated with this node
inTrans- The transform information to store in the node. Usually this is some offset from the parent's transform.
