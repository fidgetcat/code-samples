Detailed Description

Base class for the game's event handlers.

This engine processes events instead of having predefined code in an update loop. Objects that trigger or listen for events will normally inherit from either TimeHandler or Keyhandler, and then implement the abstract functions. KeyHandler has an enum interface that works with SFML, while the Engine has a clock that works for TimeHandler. Objects that want to be interacted with must inherit from or have an InteractHandler. To be drawn by the engine, objects must be DrawHandlers.
